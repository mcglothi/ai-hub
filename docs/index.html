Get AI Hub running

Designed to run on a dedicated VM or any always-on Linux host on your local network.

git clone https://github.com/mcglothi/ai-hub.git cd ai-hub && docker compose up -d
Web UI at http://<host>:3001 after startup
git clone https://github.com/mcglothi/ai-hub.git cd ai-hub && npm install cp .env.example .env # edit API keys npm start
Requires Node.js 18+, ttyd, and code-server on PATH
